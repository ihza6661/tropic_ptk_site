## üö® Red Flags to Avoid

### **Things That Will Hurt Your Rankings:**

- [ ] **Keyword stuffing:** Don't add keywords to business name
  - ‚ùå Bad: "Tropic Coffee 24 Jam WiFi Musholla Pontianak"
  - ‚úÖ Good: "Tropic di Merdeka"

- [ ] **Inconsistent NAP:** Name, Address, Phone must match website/social media exactly

- [ ] **Duplicate listings:** One location = one profile only

- [ ] **Stock photos:** Only use real photos of YOUR location

- [ ] **Ignoring reviews:** Respond to ALL reviews within 48h

- [ ] **Fake reviews:** NEVER buy reviews or incentivize them dishonestly

- [ ] **Sparse content:** Empty profiles rank lower (add photos, posts, Q&A)

- [ ] **Wrong category:** Primary MUST be accurate ("Coffee Shop" not generic "Restaurant")

- [ ] **Outdated info:** Keep hours, menu, photos current

- [ ] **No engagement:** Dead profiles (no posts, no responses) rank lower

## üìû Support & Troubleshooting

### **Common Issues:**

**Issue:** Profile suspended
- **Solution:** Contact Google support, ensure compliance with guidelines

**Issue:** Unable to verify
- **Solution:** Request new postcard, try phone verification, contact support

**Issue:** Low visibility
- **Solution:** Add more photos, get more reviews, post regularly, optimize description

**Issue:** Duplicate listings
- **Solution:** Report duplicates, merge if owned by you, contact support

**Issue:** Wrong information showing
- **Solution:** Edit profile, wait 24-48h for update, suggest edits if others changing it

### **Getting Help:**
- Google Business Profile Help: https://support.google.com/business
- Phone support: Available in GBP dashboard (verified businesses)
- Community forum: https://support.google.com/business/community
